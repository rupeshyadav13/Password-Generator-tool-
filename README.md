# Password Generator Tool

A simple and secure Password Generator Tool that creates strong random passwords to improve online security. This tool allows users to generate passwords with customizable length and character sets, including uppercase letters, lowercase letters, numbers, and special characters.

# Features
Generate strong and random passwords
Customizable password length
Support for:
Uppercase letters (A-Z)
Lowercase letters (a-z)
Numbers (0-9)
Special characters (!@#$%^&*)
Easy-to-use interface
Lightweight and fast

# Benefits

Enhances account security
Reduces the risk of password guessing and brute-force attacks
Saves time by automatically creating complex passwords

# Technologies Used

Python
Random module
Command Line Interface (CLI)

# How to Use

Run the program.
Enter the desired password length.
Select the character types to include.
Click Generate or execute the command.
Copy and use the generated password securely.

# Example Output

Plain text
Generated Password: A#7mK@9qP$2x
